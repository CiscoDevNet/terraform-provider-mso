# mso_schema #

Data source for MSO schema  

## Example Usage ##

```hcl
data "mso_schema" "sample_schema" {
  name  = "schema1"
}
```

## Argument Reference ##

* `name` - (Required) name of the schema.

## Attribute Reference ##

* `template_name` - (Optional) name of template Attached to this schema.
* `tenant_id` - (Optional) tenant_id for this schema.
